## рІерїерІІрЅ│ рЅБрѕЁрѕфрІФрЅх

### РюЁ **рѕЎрѕЅ рЅБрѕЁрѕфрІФрЅх**
- **рІерѕёріГрѕ│рїјріЊрѕЇ рІЊрѕѕрѕЮ рѕхрѕГрІЊрЅх** - ріерЅБрІ«рѕЮ рїІрѕГ рЅарЅ░рѕФрЅЂ рІЊрѕѕрѕЏрЅх
- **рѕўрЇЇрїерЅх ріЦріЊ рѕўрѕхрѕФрЅх** - ріерЅ░рѕѕрІФрІЕ рІеріЋрїЦрѕе ріљрїѕрѕГ рЇЇрїЦріљрЅх рїІрѕГ рІерѕўрѕ│рѕфрІФ рѕІрІГ рІерЅ░рѕўрѕарѕерЅ░ рѕўрЇЇрїерЅх
- **рІерЅЦрѕјріГ рѕЏрЅђріЊрЅарѕфрІФ** - рІерѕџрЅђріЊрЅарѕГ рЅЦрѕјріГ рѕѕрѕЏрѕхрЅђрѕўрїЦ рЅђріЮ рїарЅЁ рІФрІхрѕГрїЅ
- **рІерібріЋрЅгріЋрЅХрѕф рѕхрѕГрІЊрЅх** - ріерѕєрЅхрЅБрѕГ рїІрѕГ 32-рѕхрѕјрЅх рібріЋрЅгріЋрЅХрѕф (9 рѕхрѕјрЅх)
- **рІерїдрѕГ рѕхрѕГрІЊрЅх** - ріерІерѕЮрѕЇрїарїФ ріЦріЋрїЇрІ│ рїІрѕГ рІерїцріЊ/рїЅрІхрѕѕрЅх рѕхрѕГрІЊрЅх
- **рІерЅђріЋ/рѕїрѕірЅх рІЉрІ░рЅх** - рЅ░рѕѕрІІрІІрїГ рЅЦрѕГрѕЃріЋ ріЦріЊ рІерїірІю рѕѓрІ░рЅх
- **рІеріарІерѕГ ріЋрЇІрѕх рЅ░рїйріЦріќрІјрЅй** - рІерІЮріЊрЅЦрЇБ рІерЅарѕерІХ ріЦріЊ рІеріарЇІрѕГ рѕхрѕГрІЊрЅХрЅй
- **рІерѕЏрѕхрЅђрѕўрїФ/рІерѕЏрѕхрїѕрЅБ рѕхрѕГрІЊрЅх** - ріарІЇрЅХрѕЏрЅ▓ріГ рѕЏрѕхрЅђрѕўрїФ рІФрѕѕрІЇ рІерІЊрѕѕрѕЮ рѕЂріћрЅ│

### ­Ъј« **рѕўрЅєрїБрїарѕфрІФрІјрЅй**
- **WASD / рЅђрѕхрЅХрЅй**: ріЦріЋрЅЁрѕхрЅЃрѕ┤
- **ріГрЇЇрЅх рЅдрЅ│**: рѕўрѕФрѕўрІх / рѕўрІФрІЮ
- **рІерїЇрѕФ рїарЅЁ**: рІерЅЦрѕјріГ рѕўрЇЇрїерЅх
- **рІерЅђріЮ рїарЅЁ**: рІерЅЦрѕјріГ рѕЏрЅђріЊрЅарѕфрІФ
- **E**: рІерѕўрѕхрѕФрЅх рѕюріЉ ріГрЇѕрЅх
- **Q**: рІерЅ░рѕФрЅЂ ріЦріЋрїѕрЅЦ рѕѕрѕўрѕхрЅђрѕЇ
- **рІеріарІГрїЦ рїѕрѕўрІх**: рІерѕєрЅхрЅБрѕГ рѕЮрѕГрїФ
- **1-9**: рЅђрїЦрЅ│ рІерѕєрЅхрЅБрѕГ рѕЮрѕГрїФ
- **F5**: рІерЅ░рІФрІФрІЦ рѕЏрѕхрЅђрѕўрїФ
- **F9**: рІерЅ░рІФрІФрІЦ рѕЏрѕхрїѕрЅБ
- **ESC**: рѕюріЉ / рѕюріЉрІјрЅй рІГрІЮ

## рѕўрїЇрїерЅх ріЦріЊ рѕЏрІІрЅђрѕГ

### рІФрѕѕрЅарЅх ріљрїѕрѕ«рЅй
- **Go 1.19+** - рІІріЊ рѕърЅ░рѕГ
- **Git** - рІерѕхрѕфрЅх рѕўрЅєрїБрїарѕфрІФ

### рЇѕрїБріЋ рѕўрїђрѕўрѕфрІФ
```bash
# рІерѕЏріерѕЏрЅ╗ рЅдрЅ│ рѕѕрѕўрЅЁрІ░рЇЇ
git clone https://github.com/tesselstudio/TesselBox-game.git
cd TesselBox-game

# рїерІІрЅ│ рѕѕрѕўрѕхрѕФрЅх
go build ./cmd/client

# рїерІІрЅ│ рѕѕрѕўрѕхрѕФрЅх
./client
```

## рІерѕхрѕГрІЊрЅх рѕўрѕхрЇѕрѕГрЅХрЅй

- **CPU**: рІерѕЂрѕѕрЅх рі«рѕГ рЇЋрѕ«рѕ░рѕ░рѕГ

- **GPU**: ріеOpenGL 3.3+ рїІрѕГ рІерѕџрѕхрѕЏрѕЏ
- **рѕЏріерѕЏрЅ╗**: 500MB ріљрї╗ рЅдрЅ│

### рІерѕџрІФрѕхрЅаріЊрѕЇ
