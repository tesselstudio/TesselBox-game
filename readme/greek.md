# TesselBox - Ελληνικά README
## Εξακτινωτό Voxel Παιχνίδι

[![Open Source Helpers](https://www.codetriage.com/tesselstudio/tesselbox-game/badges/users.svg)](https://www.codetriage.com/tesselstudio/tesselbox-game)

Ένα παιχνίδι περιπέτειας sandbox 2D εμπνευσμένο από το *Terraria*, αλλά χτισμένο σε **εξακτινωτό πλέγμα**.

Εξερευνήστε κόσμους, εξορύξτε πόρους, χτίστε δομές, δημιουργήστε αντικείμενα, πολεμήστε εχθρούς και επιβιώστε — όλα σε όμορφα εξακτινωτά πλακίδια.

## Χαρακτηριστικά Παιχνιδιού

### ✅ **Πλήρη Χαρακτηριστικά**
- **Εξακτινωτή Δημιουργία Κόσμου** - Διαδικαστικά δημιουργημένοι κόσμοι με βιότοπα
- **Εξόρυξη και Κατασκευή** - Εξόρυξη βασισμένη σε εργαλεία με διαφορετικές ταχύτητες υλικών
- **Τοποθέτηση Μπλοκ** - Δεξί κλικ για τοποθέτηση μπλοκ με προεπισκόπηση φάντασμα
- **Σύστημα Αποθέματος** - Απόθεμα 32 θέσεων με γρήγορη γραμμή (9 θέσεις)
- **Σύστημα Μάχης** - Σύστημα υγείας/ζημιάς με κινήσεις επίθεσης
- **Κύκλος Ημέρα/Νύχτα** - Δυναμικός φωτισμός και πρόοδος χρόνου
- **Εφέ Καιρού** - Συστήματα βροχής, χιονιού και καταιγίδας
- **Σύστημα Αποθήκευσης/Φόρτωσης** - Μόνιμη κατάσταση κόσμου με αυτόματη αποθήκευση

### 🎮 **Χειρισμός**
- **WASD / Βέλη**: Κίνηση
- **Διάστημα**: Άλμα / Επίθεση
- **Αριστερό Κλικ**: Εξόρυξη μπλοκ
- **Δεξί Κλικ**: Τοποθέτηση μπλοκ
- **E**: Άνοιγμα μενού κατασκευής
- **Q**: Απόθεση επιλεγμένου αντικειμένου
- **Τροχός Ποντικιού**: Επιλογή γρήγορης γραμμής
- **1-9**: Άμεση επιλογή γρήγορης γραμμής
- **F5**: Χειροκίνητη αποθήκευση
- **F9**: Χειροκίνητη φόρτωση
- **ESC**: Μενού / Κλείσιμο μενού

## Εγκατάσταση και Ρύθμιση

### Προαπαιτούμενα
- **Go 1.19+** - Κύρια μηχανή
- **Git** - Έλεγχος έκδοσης

### Γρήγορη Έναρξη
```bash
# Κλωνοποίηση αποθετηρίου
git clone https://github.com/tesselstudio/TesselBox-game.git
cd TesselBox-game

# Δημιουργία παιχνιδιού
go build ./cmd/client

# Εκτέλεση παιχνιδιού
./client
```

### Ρύθμιση Ανάπτυξης
```bash
# Εγκατάσταση εξαρτήσεων
go mod tidy

# Εκτέλεση δοκιμών
go test ./...

# Δημιουργία για ανάπτυξη
go build -tags debug ./cmd/client
```

## Απαιτήσεις Συστήματος

### Ελάχιστες
- **Λειτουργικό Σύστημα**: Windows 10+, macOS 10.15+, Linux
- **Επεξεργαστής**: Διπύρηνος επεξεργαστής
- **RAM**: 4GB
- **GPU**: Συμβατό με OpenGL 3.3+
- **Αποθήκευση**: 500MB ελεύθερος χώρος

### Συνιστώμενες
- **Επεξεργαστής**: Τετραπύρηνος επεξεργαστής
- **RAM**: 8GB+
- **GPU**: Αφιερωμένη κάρτα γραφικών
- **Αποθήκευση**: 1GB+ ελεύθερος χώρος

## Αρχιτεκτονική

### Βασικές Τεχνολογίες
- **Γλώσσα**: Go (Golang)
- **Γραφικά**: Ebiten (βιβλιοθήκη παιχνιδιών 2D)
- **Σύστημα Δημιουργίας**: Άρθρωμα Go

### Δομή Έργου
```
TesselBox/
├── cmd/client/          # Κύριο εκτελέσιμο αρχείο παιχνιδιού
├── pkg/                 # Βασικά πακέτα
│   ├── world/          # Δημιουργία και διαχείριση κόσμου
│   ├── player/         # Μηχανισμοί παίκτη και φυσική
│   ├── blocks/         # Τύποι μπλοκ και ιδιότητες
│   ├── items/          # Σύστημα αντικειμένων και κατασκευή
│   ├── crafting/       # Συνταγές κατασκευής και διεπαφή
│   ├── weather/        # Προσομοίωση καιρού
│   ├── gametime/       # Κύκλος ημέρα/νύχτα
│   ├── save/           # Λειτουργία αποθήκευσης/φόρτωσης
│   └── render/         # Συστήματα απόδοσης και διεπαφής
├── config/             # Αρχεία διαμόρφωσης
└── assets/             # Στοιχεία παιχνιδιού (αν υπάρχουν)
```

## Συνεισφορά

### Για Προγραμματιστές
1. Κάντε fork το αποθετήριο
2. Δημιουργήστε κλάδο χαρακτηριστικού (`git checkout -b feature/amazing-feature`)
3. Κάντε commit τις αλλαγές σας (`git commit -m 'Add amazing feature'`)
