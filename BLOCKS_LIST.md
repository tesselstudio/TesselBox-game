## Humidity Implementation Summary

### **Visual Effects by Humidity Range:**
- **Dry (0.1-0.3)**: Lighter colors, clean surfaces, sparse details
- **Normal (0.3-0.7)**: Standard colors, moderate textures, normal details  
- **Wet (0.7-1.0)**: Darker colors, wet effects, lush details

### **Common Patterns Across All Blocks:**
- **Color Gradients**: Darkening in wet areas, lightening in dry areas
- **Texture Variations**: Moss, rust, slime, damp patches
- **Surface Effects**: Wet sheen, fog, frost, erosion
- **Particle Systems**: Water droplets, mist, condensation

### **Implementation Notes:**
- Each block uses 3-tier humidity system for consistent variation
- Visual effects stack (base color + humidity modifier + pattern)
- Biome humidity determines base variation level
- Local humidity can create micro-variations within biomes

This system provides rich visual variety while maintaining performance through procedural generation rather than individual textures.

## Block Properties
Each block has the following properties:
- `ID` - Internal block type identifier
- `Name` - Display name
- `Color` - Base color (RGBA)
- `TopColor` - Top face color (optional)
- `SideColor` - Side face color (optional)
- `Colors` - Color palette for patterns (optional)
- `Hardness` - Mining difficulty
- `Transparent` - Whether light passes through
- `Solid` - Whether player can stand on
- `Collectible` - Whether it drops as item
- `Flammable` - Whether it burns
- `LightLevel` - Light emission level
- `Gravity` - Whether it falls
- `Viscosity` - Liquid flow properties
- `Pattern` - Visual pattern type
- `Texture` - Custom texture file (optional)

## Biome Properties Explanation

### **Property Ranges:**
- **Tree Density**: 0.0 (no trees) to 1.0 (dense forest)
- **Ore Frequency**: Multiplier for ore generation (0.5x = half, 2.0x = double)
- **Temperature**: 0.0 (freezing) to 1.0 (extremely hot)
- **Humidity**: 0.0 (arid) to 1.0 (saturated)

### **Biome Categories:**
- **Basic Biomes**: 4 types (Plains, Forest, Desert, Mountains)
- **Water Biomes**: 4 types (Ocean, Swamp, Coral Reef, Mangrove)
- **Cold Biomes**: 3 types (Taiga, Tundra, Ice Fields)
- **Hot/Humid Biomes**: 2 types (Jungle, Savanna)
- **Special Biomes**: 1 type (Volcanic)

**Total: 14 different biome types**

### **Biome Generation:**
Biomes are determined using simplex noise based on:
- Temperature noise
- Humidity noise
- Elevation noise
- Continental noise (for larger regions)

This creates realistic, flowing biome transitions rather than sharp boundaries.
